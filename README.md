# Boilerplate: Single repo NestJS + VueJS

## getting started

#### Create backend Nestjs app :
```bash
docker-compose run backend zsh
```
```bash
nest --help
```
#### Create frontend vuejs app:

```bash
docker-compose run backend zsh
```
```bash
vue --help
```

#### Then start everyting
```bash
docker-compose up --build
```
